# shoppingCart

### Synopsis

A simple online shop with a cart.

### Installation

// open terminal
```
> mongod
```
// open a new terminal 
```
> git clone https://github.com/sjaymoon15/shoppingCart.git
> cd server
> npm install
> node index.js
```
// open a new terminal and go back to root directory then
```
> cd client
> npm install
> npm run start
```
// open a web browser and enter 
http://localhost:8080

Have fun!
